# 🎨 Thumblify – AI Thumbnail Generator

Thumblify is an AI-powered thumbnail generation platform built with the MERN stack. Users can generate custom thumbnails using text prompts and download AI-generated images through an intuitive and responsive interface.

## 🚀 Features

* AI-Powered Thumbnail Generation
* Prompt-Based Image Creation
* User Authentication & Authorization
* Thumbnail Preview & Download
* Responsive User Interface
* RESTful API Integration
* Secure User Management

## ⚙️ Installation

```bash
git clone <repository-url>
cd thumblify
```

Install dependencies:

```bash
npm install
```

Start frontend:

```bash
npm run dev
```

Start backend:

```bash
npm start
```
